# Descripción del proyecto
🌐 RedGeoScan es una herramienta avanzada diseñada para simplificar el manejo y organización de datos geodésicos. Conectada a la [Red Activa de Antenas Geodésicas de Colombia](https://www.colombiaenmapas.gov.co/?e=-74.19790397936877,4.606824129536961,-74.05594002062873,4.715639657991442,4686&b=igac&u=0&t=25&servicio=8)
, esta aplicación realiza las siguientes funciones de manera eficiente y automática:

📍 Localiza las 8 antenas más cercanas a una coordenada calculada como la media base de un archivo POS.
✅ Verifica el estado de las antenas en tiempo real, garantizando la fiabilidad de los datos.
📂 Identifica y descarga archivos RINEX disponibles, organizándolos automáticamente en carpetas estructuradas por fecha, para facilitar su análisis y acceso.
⚙️ Optimiza el flujo de trabajo en proyectos geodésicos, ahorrando tiempo y minimizando errores manuales.
🚀 RedGeoScan es tu aliado perfecto para proyectos de geodesia y geomática, integrando precisión y automatización en una solución intuitiva y eficaz.

# Tecnologías utilizadas ⚙️

🖥️ El programa **RedGeoScan** está diseñado en **Python**, un lenguaje de programación versátil y poderoso, ideal para manejar y procesar grandes volúmenes de datos geodésicos. Además, aprovecha un conjunto de **librerías especializadas** para garantizar un rendimiento óptimo y funcionalidades avanzadas:

- 📊 **Pandas**: Para la manipulación y análisis eficiente de datos tabulares.  
- 🔢 **NumPy**: Para cálculos numéricos y manejo de matrices.  
- 🌐 **Requests**: Para realizar solicitudes HTTP y manejar descargas de datos.  
- 📈 **Matplotlib** (u otra librería gráfica): Para generar gráficos y visualizaciones.  
- 🗺️ **Geopy** (o librerías de geolocalización): Para cálculos de distancia y manejo de coordenadas.
